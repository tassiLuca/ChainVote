## Requirements

- Unix system (either Linux or MacOS)
- Docker
  - on Mac OS make sure the file sharing implementation for the container is set to `osfx (Legacy)` (Settings -> General)
- `npm`

## Startup

To bring up the whole system (it will take some minutes):

```bash
./services up
```

To shutdown:
```bash
./services down
```
